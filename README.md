# Sports Betting System

Система для управления пользователями, их балансами и расчётом ставок на спортивные события.

## Функциональность

- **Клиентская часть**: просмотр событий и коэффициентов, размещение ставок, просмотр баланса с конвертацией в другие валюты, смена валюты, история операций, управление контактами.
- **Административная панель**: управление пользователями, контактами, балансами, расчёт ставок (выигрыш/проигрыш), управление курсами валют, просмотр логов изменений баланса.

### Поддерживаемые операции

- Регистрация и аутентификация пользователей
- Размещение ставок (сумма от 1 до 500 в валюте пользователя)
- Автоматический расчёт выигрыша: `ставка × коэффициент`
- Смена валюты с конвертацией баланса по текущему курсу
- Ведение логов всех изменений баланса (ставки, расчёты, корректировки админа, смена валюты)
- Несколько контактов на пользователя (до 2 телефонов и 2 email)

## Технологии

- PHP 8.4 (OOP, SOLID, DTO, PSR-4 autoloading)
- MySQL 5.7
- Nginx
- Docker / docker-compose
- HTML, CSS, JavaScript, jQuery
- Без фреймворков и сторонних PHP-зависимостей

## Запуск проекта

```bash
# Сборка и запуск всех сервисов
make up

# Или с пересборкой:
docker-compose up --build -d
```

После запуска:

| Сервис       | URL                        |
|-------------|----------------------------|
| Клиентская часть | http://localhost         |
| Админ-панель     | http://localhost/admin   |
| phpMyAdmin       | http://localhost:8080    |

## Тестирование

```bash
make test
```

Запускает 35+ тестов, покрывающих:

### База данных
- Подключение к БД
- Поиск пользователя по логину и ID
- Получение списка всех пользователей
- Получение контактов пользователя
- Добавление и удаление контактов (включая лимит 2 на тип)
- Балансы: поиск по пользователю, поиск по пользователю и валюте
- Создание и поиск ставок, поиск ставок пользователя
- Курсы валют: получение всех, получение конкретного, обновление
- Логи баланса: создание записи, поиск по пользователю

### Бизнес-логика
- Валидация суммы ставки (меньше 1, больше 500)
- Валидация коэффициентов (меньше 1.01)
- Валидация исхода (неверный исход)
- Проверка недостаточного баланса
- Валидация результата при расчёте ставки
- Списание с баланса при недостатке средств
- Пополнение баланса
- Диапазон коэффициентов событий (1.01–40.00)
- Консистентность курсов валют (обратный курс = 1 / прямой)
- DTO: создание PlaceBet, SettleBet, UpdateBalance, ChangeCurrency
